# Spec Templates

`.claude/specs/` is a reusable template and reference library for common task shapes.

It is **not** the live feature backlog.

Use this folder for:

- admin view templates
- contract feature templates
- indexer handler templates
- shared hook templates
- offline bugfix templates

Use `.plans/` for:

- active work
- backlog candidates
- ADRs
- ideas
- archived plan material

When a template becomes too specific to one historical initiative, archive it rather than keeping it in the live template library.
