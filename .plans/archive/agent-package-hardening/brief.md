# Agent Package Hardening Brief

Formalizes the existing pre-SMS agent hardening plan as an active hub. The work tightens current Telegram-bound agent handlers and services before `agent-messaging-channels` adds SMS, WhatsApp, and session-key surfaces.

This is intentionally scoped to `packages/agent` and agent documentation. New platform adapters, phone linking, and ERC-4337 session-key work remain owned by `agent-messaging-channels`.
