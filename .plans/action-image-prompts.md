# Action Image Generation Prompts

> **Purpose**: AI image generation prompts for all 22 Green Goods action cards
> **Style**: Organic botanical illustration + risograph screen-print texture
> **Format**: Landscape 3:2 (matches `object-cover` card crop at `h-40`/`h-26`)
> **Generator**: Compatible with Midjourney, DALL-E, Flux, Ideogram

---

## Shared Style Suffix

Appended to every prompt for visual consistency:

```
, organic botanical illustration with risograph screen-print texture,
hand-drawn ink linework on warm cream parchment background,
[ACCENT] as primary accent ink, subtle paper grain,
center-weighted composition, minimal negative space background,
soft muted palette, no text, no UI elements, landscape 3:2
```

### Domain Accent Colors

| Domain | Accent Token | UI Color |
|--------|-------------|----------|
| Solar | `warm amber and golden yellow` | amber |
| Agroforestry | `earthy sage green and olive` | green |
| Education | `soft cerulean blue and indigo` | blue |
| Waste | `burnt orange and terracotta` | orange |

### Design Rationale

- **Cream parchment base** — mid-tone that works in both light and dark mode
- **Hand-drawn linework** — feels human, not corporate; matches regenerative ethos
- **Domain accent as primary ink** — creates visual grouping in action grid
- **Center-weighted composition** — survives aggressive `object-cover` cropping
- **Minimal background** — reads quickly at small card sizes (104px height)
- **No text/UI** — prevents clashing with card title overlay

## Generation Tips

1. **Aspect ratio**: Generate at 3:2 landscape to match card crop
2. **Test batch**: Run one per domain first to calibrate texture intensity
3. **Text suppression**: If the generator adds text despite "no text", add `--no text` (Midjourney) or emphasize "purely visual, no words, no letters, no typography"
4. **Dark mode**: The cream parchment base acts as a natural mid-tone — won't blow out in light mode or disappear in dark mode
5. **Consistency pass**: After generating all 22, review as a grid to check visual coherence across domains
6. **File naming**: Use the action slug for filenames (e.g., `solar-site-setup.webp`, `agro-planting-event.webp`)
7. **Export format**: WebP at 80% quality, 800x533px (3:2) — balances quality with PWA bundle size
