- **File**: `packages/shared/src/hooks/work/useWorkApprovalActions.ts:52`
- **Risk score**: 3.0 (impact=1 x likelihood=3 x staleness=1.0)
- **Confidence**: HIGH
- **Issue**: The `viewingMode` parameter is destructured from the params object but never referenced in the function body. oxlint reports: "Parameter 'viewingMode' is declared but never used." The parameter exists in the `UseWorkApprovalActionsParams` interface (line 27) and is typed as `"operator" | "gardener" | "viewer"`, suggesting it was intended for gating approval submission to operators only, but this logic is not yet implemented.
- **Recommendation**: Either implement the intended operator check (e.g., early-return from `handleSubmitApproval` when `viewingMode !== "operator"`) or prefix with underscore (`_viewingMode`) if intentionally deferred.

## Refactoring Quality Assessment

The uncommitted changes represent a high-quality refactoring pass:

1. **`useSafeMutation` pattern** -- Eliminates ~60 lines of duplicated boilerplate (`useMutationLock` + `useBeforeUnloadWhilePending` + `useCallback` wrapping) from 5 hooks. Clean single-responsibility extraction.

2. **`useProgressiveInvalidation` pattern** -- Replaces single-fire `useDelayedInvalidation(fn, 2000)` with progressive schedule `[2s, 5s, 15s]`. Properly handles mount lifecycle, cleanup, and ref-based invalidation. Applied consistently across all 13 mutation hooks.

3. **Auth provider split** -- Triple context (`AuthStateContext`, `AuthActionsContext`, `AuthContext`) allows consumers to subscribe to state-only or actions-only, reducing re-renders. Backward compatible via combined `AuthContext`.

4. **useMintHypercert decomposition** -- Service actors extracted to `services/` directory with clear boundaries (build-and-sign, poll-for-receipt, register-in-signal-pool, upload-metadata, upload-allowlist). Machine wiring stays in the main hook.

5. **IPFS module split** -- Clean separation into `client.ts` (init/state), `upload.ts` (upload), `resolve.ts` (parsing/fetching), `pinata.ts` (Pinata-specific). Barrel re-export preserves API.
