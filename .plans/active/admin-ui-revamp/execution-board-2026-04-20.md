1. Claude completes a route composition change, commits it, and names the exact file when handing it off.
2. Codex pulls the latest branch state before touching the handed-off file.
3. While Codex is in a handed-off route file, Claude does not keep editing that file.
4. Lane A does not modify layout shell `.tsx` files unless Claude explicitly hands off the file for a story/test reason.
5. Lane B produces triage notes first; any actual bug fix requires Claude to re-sequence the sprint.
