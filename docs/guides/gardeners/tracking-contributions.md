## Work Statuses

### Pending ‚è≥

**What It Means**:
- Operator hasn't reviewed yet
- Waiting in queue
- Nothing wrong, just not processed yet

**Typical Wait Time**: 24-48 hours

**Actions You Can Take**:
- View submission details
- Wait patiently
- Contact operator if > 7 days

### Approved ‚úÖ

**What It Means**:
- Operator validated your work
- On-chain attestation created
- Permanently recorded
- Counts toward your portfolio

**What Happens**:
- Notification sent to you
- Work moves to "Approved" section
- Attestation link available
- Metrics added to garden total

### Rejected ‚ùå

**What It Means**:
- Work didn't meet standards
- Operator provided feedback
- Can revise and resubmit

**What To Do**:
1. Read operator feedback carefully
2. Understand what needs improvement
3. Document work better next time
4. Or resubmit with corrections

## Understanding Feedback

### Approval Feedback

**Positive Examples**:
- "Excellent documentation, great photo quality"
- "Well-measured metrics, clear evidence of work"
- "Outstanding effort, exceeded expectations"

**What It Means**:
- Keep doing what you're doing
- Operator recognized quality
- Sets standard for future work

### Rejection Feedback

**Common Reasons**:
- **Photo Issues**: "Before/after photos not clear enough"
- **Metric Problems**: "Tree count seems inflated"
- **Evidence Lacking**: "Need closer photos of planted area"
- **Wrong Action**: "This work doesn't match selected action"

**How to Respond**:
- Don't take it personally
- Learn from feedback
- Improve next submission
- Ask questions if unclear

### Requesting Clarification

**If Feedback Is Unclear**:
1. Check garden's communication channel (if any)
2. Contact operator via community channels
3. Ask for specific improvement guidance
4. Be respectful and open to learning

## Common Questions

### "Why is approval taking so long?"

**Typical reasons**:
- Operator has backlog
- Garden has few operators
- Your work needs careful review
- Seasonal delays (holidays, etc.)

**What to do**:
- Wait patiently (up to 7 days)
- Check garden's communication channels
- Politely inquire if > 7 days
- Consider helping as operator yourself!

### "Can I edit after submitting?"

**No, but**:
- Submissions are locked once sent
- Prevents gaming the system
- If rejected, resubmit corrected version
- Original stays in history for transparency

### "What if I disagree with rejection?"

**Steps**:
1. Read feedback carefully
2. Consider operator's perspective
3. If genuine error, contact operator
4. Provide additional context
5. Request re-review
6. Be professional and respectful

**Resolution**:
- Most issues resolve through communication
- Operators want to approve quality work
- Consensus usually reached
- If persistent issues, escalate to garden admin

### "Can I delete submissions?"

**Short answer**: No

**Why**:
- Maintains transparency
- Historical record
- Prevents abuse
- All submissions contribute to learning

**Privacy**:
- Contact support for legitimate privacy concerns
- Data is pseudonymous (wallet address only)
- Photos are your submissions
