## Step 2: Fill Details

### Open the Submission Form

1. Tap "Submit Work" from home or garden view
2. Or continue from "Start Work" flow

### Required Fields

**Select Action** üéØ

Tap the dropdown and choose which action you completed:
- Actions are pre-defined by garden operators
- Read the description to confirm match
- Each action has specific metrics

**Title** (Auto-Generated)

Leave blank to use action title, or customize:
- Brief description of what you did
- Example: "Planted 15 oak trees in north section"
- Keep it under 100 characters

### Add Metrics

**Action-Specific Numbers** üìä

Different actions require different metrics. Common ones:

**For Tree Planting**:
- Number of trees: 15
- Species: Quercus agrifolia
- Area covered: 300 sqm

**For Litter Cleanup**:
- Bags collected: 4
- Area covered: 500 sqm
- Type of waste: Plastic, paper, metal

**For Monitoring**:
- Species count: 12
- Individuals observed: 47
- Time spent: 2 hours

**Tips**:
- Count carefully (don't estimate)
- Use consistent units
- Be honest about quantities
- Note unusual conditions

### Location

**GPS Auto-Detection** üìç

- App detects your location automatically
- Verify it's correct on the map
- Adjust pin if needed
- Or enter location manually

**Privacy**:
- Location helps map impact
- Only operators see exact coords
- Public views show general area

### Notes/Context (Optional)

**Add Additional Info** ‚úçÔ∏è

- Challenges encountered
- Observations worth sharing
- Collaborators involved
- Weather conditions
- Why this location matters

**Example**:
```
Planted trees on steep slope requiring extra care with erosion control. Used burlap wrapping for root protection. Local elementary school volunteered 10 students to help.
```

## Step 3: Review Submission

### Preview Screen

**What You'll See**:
- ‚úÖ All uploaded photos (in gallery view)
- ‚úÖ Action title and description
- ‚úÖ All metrics you entered
- ‚úÖ Location (map pin)
- ‚úÖ Notes/context

**Check Everything**:
- Photos uploaded correctly (not blurry)
- Metrics are accurate
- Location is correct
- Spelling is correct

### Edit if Needed

**Can Still Change**:
- Go back to edit any section
- Upload different photos
- Adjust metrics
- Add forgotten details

**Cannot Change After Submission**:
- Once submitted, work is locked
- Operator reviews what you sent
- If rejected, can resubmit new version

### Submit

**Tap "Submit Work"** üöÄ

**If Online**:
1. Photos upload to IPFS (~5-30 seconds)
2. Transaction prepared
3. Smart account submits attestation
4. Confirmation shown
5. Work appears in "Pending" status

**If Offline**:
1. Work saved to local queue
2. "Queued for sync" status shown
3. Auto-uploads when online
4. You'll see confirmation later

## Common Issues

### "Photo won't upload"

**Solutions**:
- Check file size (< 10 MB recommended)
- Try different photo
- Check camera permissions
- Clear app cache

### "Can't find my action"

**Solutions**:
- Verify you're in the correct garden
- Ask operator if action is active
- Refresh garden actions list
- Check if action has time window

### "Location detection failed"

**Solutions**:
- Enable location permissions
- Try manual pin placement
- Enter location text description
- Submit without location (if optional)

### "Submission stuck in queue"

**Solutions**:
- Check internet connection
- Wait for retry (auto-retries every 5 min)
- Force sync from Work Dashboard
- Contact support if persistent

## Best Practices

### Photo Quality

‚úÖ **Do**:
- Clear, well-lit images
- Same angle for before/after
- Show full work area
- Include context/landmarks

‚ùå **Don't**:
- Blurry or dark photos
- Different angles for before/after
- Overly zoomed in
- Apply filters

### Accurate Metrics

‚úÖ **Do**:
- Count carefully
- Use standard units
- Be honest about quantities
- Note estimation if necessary

‚ùå **Don't**:
- Guess or inflate numbers
- Use inconsistent units
- Round excessively
- Omit important context

### Timely Submission

‚úÖ **Do**:
- Submit same day as work
- Document in the field
- Use offline mode if needed
- Keep photos organized

‚ùå **Don't**:
- Wait weeks to submit
- Submit old work without context
- Batch unrelated work together
- Rush documentation

## Advanced Tips

### Documenting Long-Term Work

**For Multi-Day Projects**:
- Submit each day's work separately
- Reference previous submissions
- Track progress over time
- Use consistent photo locations

### Collaborative Work

**When Working as Team**:
- Each gardener submits own work
- Or designate one person to document
- Note collaborators in context
- Share responsibilities fairly

### Seasonal Patterns

**Adapt to Conditions**:
- Planting season: Focus on installation
- Growing season: Monitor and maintain
- Harvest season: Document yield
- Winter: Preparation and planning
