# Weekly metrics digests

Written automatically by the `gg-data-analyst` routine (see `routines/gg-data-analyst.md`).

Filename convention: `YYYY-WW.md` where `WW` is ISO week number (1–53).

Example: `2026-15.md` = week 15 of 2026 (mid-April).

Each digest is opened as a PR from `claude/data-analyst/YYYY-WW` → `develop`, labeled `automated/claude-routine`. Review, merge to develop, then include in the next `develop → main` batch promotion.
