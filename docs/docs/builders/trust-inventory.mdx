# Builders Trust Inventory

This page is intentionally unlisted. It is a working trust inventory for builder documentation, not public guidance.

The table records the current status label, visibility, source posture, main trust risk, and recommended maintenance action for each builder page. It is based on frontmatter, `source_of_truth`, the current docs audit rules, and the review findings addressed in this trust pass.

## Maintenance Rules

- Listed `Live` and `Implemented` pages should stay source-backed.
- External pages should not make Green Goods usage claims without a local source.
- Draft, generated, proposal, or private-operational pages stay unlisted until a tracked source artifact exists.
- Deployment and integration pages must keep deployment artifacts, indexer config, and UI readiness separate.
