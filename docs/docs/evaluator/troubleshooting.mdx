## Common issues

<FeatureState
  title="No records returned"
  status="Live"
  summary="Usually chain/schema mismatch or wrong recipient address in variables."
>
  Re-check endpoint chain, schema UID source, and recipient format.
</FeatureState>

<FeatureState
  title="Metadata exists but v1 module events missing"
  status="Implemented (activation pending deployment)"
  summary="Indexer config has zero-address modules until deployment activation."

  Confirm non-zero addresses in deployment artifacts and indexer config.
</FeatureState>

<FeatureState
  title="Inconsistent joins"
  status="Live"
  summary="Work and approval datasets can be joined incorrectly if ref UID logic is ignored."
>
  Join on explicit attestation UID/ref UID pairs, not only action IDs.
</FeatureState>

## Escalation data

1. Query text and variables.
2. Chain ID + endpoint URL.
3. Deployment artifact file used.
4. Sample missing UID(s).
