## Suggested mapping dimensions

- Action domain/capital to SDG/GAP category.
- Evidence quality score from review confidence.
- Chain and attestation UID for auditability.
- Time windows for periodic reporting.

## Output contract

For each mapped record include:

1. Source garden/action/work identifiers.
2. External framework key.
3. Human-readable rationale.
4. Attestation UID evidence pointer.
