## Dataset minimum columns

- `chain_id`, `garden_address`, `action_uid`
- `work_attestation_uid`, `approval_attestation_uid`
- `approved`, `submitted_at`, `approved_at`
- `framework_tags`, `source_query_version`

## Reproducibility checklist

1. Save exact GraphQL query text.
2. Save variables used for each run.
3. Save deployment artifact commit hash.
4. Save transform script version.
