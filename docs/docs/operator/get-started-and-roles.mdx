## Overview

Operators are the trusted coordinators who manage gardens and validate gardener submissions. You use the admin dashboard to review work, manage membership, create assessments, and oversee garden finances.

## Best Practices

- Review submissions promptly — gardeners depend on timely feedback to stay engaged
- When rejecting work, provide specific, actionable feedback so the gardener knows exactly what to fix
- Keep your garden actions up to date with clear instructions and current time windows
- Document your garden's review standards so other operators can maintain consistency
