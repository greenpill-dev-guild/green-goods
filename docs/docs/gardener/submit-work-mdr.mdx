### Field tips

1. Save progress before leaving field conditions — drafts persist in your browser's local storage.
2. Reopen drafts and verify media is still attached before submitting.
3. Check queue/upload states before repeating submissions — duplicate submissions waste operator review time.
4. If a job fails repeatedly, retry from your work dashboard with corrected data.
