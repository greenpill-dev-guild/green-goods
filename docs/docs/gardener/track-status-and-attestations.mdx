## Status meanings

- **Pending:** Waiting for operator review.
- **Approved:** Work validated; approval attestation exists.
- **Rejected:** Work requires correction and resubmission.
- **Uploading/Queued:** Submission is still syncing.

## Attestation verification workflow

1. Open approved work details.
2. Copy/select attestation UID and chain.
3. Verify on the matching EAS explorer endpoint.
4. Confirm approval linkage and timestamps.
