## What you can do now

1. Sign in with passkey (Face ID / Touch ID / device biometrics).
2. Open an action in a garden.
3. Submit work through MDR (Media -> Details -> Review).
4. Keep working offline and sync later.
5. Track pending/approved/rejected work states.
