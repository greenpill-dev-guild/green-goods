## Non-negotiables

- Hooks live in `packages/shared`.
- Single chain derives from `VITE_CHAIN_ID` defaults.
- Deployment changes go through `bun deploy:*` wrappers.
- User-facing strings must be added to `en/es/pt` translation files.
