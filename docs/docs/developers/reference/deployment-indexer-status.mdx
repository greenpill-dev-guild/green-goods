<QuickAnswer question="How should we label features in user docs?" openByDefault>
  Use these labels only:
  <ul>
    <li><code>Live</code>: deployed and indexed in the target chain context.</li>
    <li><code>Live (external source)</code>: data intentionally comes from external SDKs/APIs/RPC, not Envio.</li>
    <li><code>Implemented (activation pending indexing)</code>: deployed but indexer is not active yet.</li>
    <li><code>Implemented (activation pending deployment)</code>: code exists but deployment is not active yet.</li>
    <li><code>Planned</code>: not implemented in code.</li>
  </ul>
</QuickAnswer>
