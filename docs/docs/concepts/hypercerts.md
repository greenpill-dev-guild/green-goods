## Hypercert Lifecycle in Green Goods

### 1. Accumulate Verified Work

Gardens track cumulative approved work over time:
- All work has on-chain attestations
- Operators validated quality
- Metrics aggregated to garden level

### 2. Mint Hypercert

At quarterly or annual milestones:
```
Garden mints Hypercert with:
- Scope: Conservation activities completed
- Time: Period covered (e.g., Q1 2024)
- Work: Links to all attestation UIDs
- Contributors: Weighted by contribution
```

### 3. Fractionalize

Hypercert is split into shares:
- 70% to active gardeners (weighted by approved work)
- 20% to operators (validation work)
- 10% to garden treasury (coordination)

### 4. Market Listing

Hypercert fractions can be:
- Listed on impact marketplaces
- Purchased by aligned funders
- Traded between parties
- Held for long-term value

### 5. Retroactive Funding

When funders purchase Hypercert fractions:
- Value flows to contributors
- Gardeners receive direct compensation
- Operators get validation rewards
- Garden treasury grows for future work

## Attribution & Contribution Tracking

Green Goods' attestation system provides **perfect attribution** for Hypercerts:

### Contribution Weighting

**For Gardeners**:
```
Weight = (Approved work count * Quality score) / Total garden work
```

**For Operators**:
```
Weight = (Reviews completed * Consistency score) / Total reviews
```

**For Garden**:
```
Weight = Fixed % for coordination overhead
```

### Example Calculation

```
Garden with 100 approved work submissions:
- Alice: 25 submissions â†’ 25% of gardener pool
- Bob: 15 submissions â†’ 15% of gardener pool
- Others: 60 submissions â†’ 60% of gardener pool

If gardener pool = 70% of total Hypercert:
- Alice gets: 25% * 70% = 17.5% total
- Bob gets: 15% * 70% = 10.5% total
```

## Multi-Capital Hypercerts

Green Goods' eight forms of capital framework enables **nuanced impact claims**:

Instead of generic "reforestation" Hypercert:

- Scope: Community engagement
- Metrics: Workshops held, participants

- Metrics: Indigenous knowledge documented

**Benefit**: Funders can support specific capital types aligned with their mission.
