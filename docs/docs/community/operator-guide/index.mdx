# Operator Guide

<StatusBadge status="Live" />

## What this helps you do

This is the core operator path. Use it to stand up a garden, define the assessment frame or baseline, approve incoming work against that frame, and then mint an impact certificate when the evidence is ready.

The key permission boundary is simple: work approval requires garden owner/operator authority. Evaluators can make assessments, but they do not approve or reject work unless they also have the operator or owner role.

## Good operating rhythm

1. Create the garden and confirm its domains before inviting a wider group.
2. Define the assessment frame early so the review queue has a clear baseline.
3. Approve or reject submissions with feedback that maps back to that frame.
4. Mint an impact certificate only after the evidence chain is strong enough to defend.
