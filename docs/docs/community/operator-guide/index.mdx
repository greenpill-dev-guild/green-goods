## What this helps you do

This is the core operator path. Use it to create a garden, make an assessment, review incoming work, and mint an impact certificate when the evidence is ready.

The key permission boundary is simple: owners, operators, and evaluators can make assessments; only owners and operators approve or reject work.

## Good operating rhythm

1. Create the garden and confirm its domains and eligible actions before inviting a wider group.
2. Make an assessment early so operators have clear context for later reviews.
3. Approve or reject submissions with feedback that maps back to the action and assessment.
4. Mint an impact certificate only after the evidence chain is strong enough to defend.
