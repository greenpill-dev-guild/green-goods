# Operator Guide

<StatusBadge status="Live" />

## What this helps you do

This is the operator path for Earth Week. Use it to stand up a garden, approve or reject incoming work, and turn approved work into an assessment.

The key permission boundary is simple: work approval requires garden owner/operator authority. Evaluators can make assessments, but they do not approve or reject work unless they also have the operator or owner role.

## Good operating rhythm

1. Keep the pending queue short so gardeners get quick feedback.
2. Reject weak submissions with specific instructions instead of vague comments.
3. Make assessments only after the approved work tells a coherent story.
