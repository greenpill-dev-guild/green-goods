## What this helps you do

This is the short path for gardeners: join the right garden, then submit work with clear evidence so an operator can review it.

You do not need to read every gardener page to get started. These two flows are the ones that matter first.

## Good first session

1. Sign in with the same account you plan to keep using.
2. Join the garden where your work belongs.
3. Open an active action before you start uploading media.
4. Submit one complete work record and check that it lands in your Work Dashboard.

## After that

Once your first submission is pending, wait for the operator review. If it is rejected, open the feedback, fix the issue, and submit a corrected version against the same action.
