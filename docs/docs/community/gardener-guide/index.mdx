## Good first session

1. Sign in with the same account you plan to keep using.
2. Join the garden where your work belongs.
3. Pick the active action that matches the work you completed.
4. Submit one complete work record and check that it lands in your Work Dashboard.

## After that

Once your first submission is pending, wait for the operator review. If it is rejected, open the feedback, fix the issue, and submit a corrected version against the right garden and action.
