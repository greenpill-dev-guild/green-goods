# Managing Gardens

Complete guide for operators to create, configure, and maintain gardens.

---

## Creating a New Garden (Admin Only)

**Prerequisites**: Admin allowlist access

**Steps**:
1. Visit [admin.greengoods.app](https://admin.greengoods.app)
2. Connect wallet
3. Click "Create Garden"
4. Fill required fields:
   - Name
   - Description
   - Location
   - Banner image (IPFS CID or upload)
5. Add initial members (gardeners/operators)
6. Submit transaction
7. Garden NFT minted + Karma GAP project created

[See Operator Quickstart →](../../welcome/quickstart-operator)

---

## Editing Garden Metadata

**Operators Can Update**:
- Description
- Banner image
- Location
- Contact information

**Cannot Change**:
- Garden name (immutable)
- Token ID
- Owner address

**Process**:
1. Navigate to garden page
2. Click "Edit Garden"
3. Modify fields
4. Save changes (transaction required)

---

## Managing Members

### Adding Gardeners

1. Garden page → "Members" tab
2. Click "Add Gardener"
3. Enter wallet address
4. Confirm transaction
5. Gardener can now submit work

### Adding Operators

1. "Members" tab → "Add Operator"
2. Enter wallet address
3. Confirm transaction
4. New operator has validation rights

### Removing Members

1. Find member in list
2. Click "Remove"
3. Confirm action
4. Member loses permissions

**Best Practices**:
- Vet members before adding
- Remove inactive contributors
- Maintain 2-3 operators minimum
- Document removal reasons
