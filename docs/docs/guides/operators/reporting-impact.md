# Reporting & Impact

Generate reports, track metrics, and communicate impact to funders.

---

## Garden Dashboard Analytics

**Key Metrics**:
- Total work submissions
- Approval rate %
- Active gardeners
- Cumulative impact (trees, area, etc.)
- Member activity trends

**Access**: Admin dashboard → Your garden → Overview

---

## Exporting Data

**For Grant Applications**:
1. Garden page → Export
2. Choose format (CSV/JSON)
3. Select date range
4. Download

**Includes**:
- All approved work
- Gardener contributions
- Metrics aggregated
- Attestation UIDs
- Photos (IPFS links)

---

## Karma GAP Integration

**Automatic Reporting**:
- Garden creation → GAP Project attestation
- Work approval → GAP Impact attestation
- Queryable via Karma GAP platform

**Benefits**:
- Standardized format
- Multi-chain reporting
- Transparent accountability
- Funder-friendly

[Karma GAP Technical Details →](../../developer/karma-gap.md)

---

## Creating Assessments

**Comprehensive Garden Evaluations**:
1. Navigate to garden
2. Click "Create Assessment"
3. Fill assessment form:
   - Title & description
   - Assessment type
   - Capitals measured
   - Metrics (JSON or form)
   - Evidence media
   - Link to work attestations
4. Submit

**Use For**:
- Quarterly reports
- Annual reviews
- Milestone celebrations
- Funder updates
