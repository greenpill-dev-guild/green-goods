# Creating & Managing Actions

Define tasks for gardeners to complete and track progress.

---

## Creating Actions

**Required Fields**:
- **Title**: Clear, specific (e.g., "Plant 10+ Native Trees")
- **Instructions**: Detailed requirements
- **Start/End Time**: Optional time window
- **Capitals**: Which forms of capital (Living, Social, etc.)
- **Media**: Example photos

**Best Practices**:
- Clear success criteria
- Measurable outcomes
- 2-4 hour scope
- Include example photos

---

## Action Templates

**Conservation**:
```
Title: Remove Invasive Species
Instructions: Remove blackberry or ivy from designated area. 
Document area before/after. Minimum 100 sqm.
Capitals: LIVING, MATERIAL
```

**Monitoring**:
```
Title: Biodiversity Survey
Instructions: Document 10+ species with photos and location. 
Include scientific names when possible.
Capitals: LIVING, INTELLECTUAL
```

**Community**:
```
Title: Lead Workshop
Instructions: Conduct environmental education session. 
Minimum 5 participants. Submit group photo and summary.
Capitals: SOCIAL, CULTURAL
```

---

## Managing Active Actions

- Activate/deactivate as needed
- Update based on season
- Archive completed campaigns
- Track completion rates
