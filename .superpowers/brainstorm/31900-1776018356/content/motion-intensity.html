Motion & Animation Intensity

How much motion should the admin dashboard have? This affects page transitions, card interactions, sheet openings, and micro-interactions.

A

Functional Motion

Springs only where they communicate state change — sheet open/close, view transitions, FAB reveal. Cards have subtle hover lift. No decorative animation. ~200-400ms durations. Feels like: Linear, Notion.

Sheet springs View crossfades Hover lifts Stagger lists Parallax
B

Expressive Motion

Everything in Functional, plus staggered list entrances, canvas recession on sheet open, card flip/expand for detail views, and smooth tab indicator sliding. ~300-500ms spring durations. Feels like: Apple apps, Airbnb.

Sheet springs View crossfades Hover lifts Stagger lists Canvas recession Tab sliding Parallax Morphing
C

Cinematic Motion

Everything in Expressive, plus shared-element transitions (card → detail morphing), parallax depth on scroll, color scheme cross-dissolve when switching views, and gestural spring physics. ~400-700ms. Feels like: visionOS, iOS 26, Arc Browser.

Note: All options respect prefers-reduced-motion. Cinematic motion requires the View Transitions API (Chrome 111+, Safari 18+).
