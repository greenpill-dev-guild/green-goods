Spatial Animation Model

Horizontal Axis (CenterContainer)
  • LeftSheet slides in from left edge
  • RightSheet slides in from right edge
  • MainSheet compresses/recedes to make room
  • Spring easing: cubic-bezier(0.16, 1, 0.3, 1)
Vertical Axis (Top/Bottom Containers)
  • TopContainer elements can animate down into view
  • BottomContainer elements can animate up into view
  • NavigationBar & FAB persist (always in view)
  • Mobile BottomSheet animates up from below viewport
