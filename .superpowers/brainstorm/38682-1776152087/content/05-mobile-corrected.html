Client ModalDrawer Reference

The client package's ModalDrawer already implements this pattern: 85dvh max height, drag handle, scrim overlay (black/30 + 4px blur), slide-up spring animation (0.4s with overshoot), focus trap, Escape dismiss. Same component powers both the EndowmentDrawer and NotificationCenter.
