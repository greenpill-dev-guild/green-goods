Design Section 1: Layout & Spatial Foundation

Sheet widths, MainSheet gutters, and bottom padding — the spatial skeleton everything else sits on.

1a. Responsive Sheet Width Tokens

Replace the hardcoded clamp(220px, 20vw, 300px) in RightSheet with CSS custom properties on the canvas grid:

CSS Custom Properties (theme.css)
--canvas-right-sheet-width: clamp(320px, 28vw, 480px);
--canvas-right-sheet-min: 320px;
--canvas-right-sheet-max: 480px;

Before: 220–300px (content clipped at every viewport)
After: 320–480px, scaling at 28vw (a 1440px screen gets ~400px, a 2560px screen gets 480px cap)

1b. MainSheet Horizontal Gutters + Max-Width

Before: Full-width flush
Content flush to edges — no sheet feeling
After: Guttered with max-width
px-4 gutter, max-w-[1400px], centered

MainSheet gets mx-4 (matching TopContextBar's px-4) and a max-w-[1400px] mx-auto for ultrawide screens. The rounded corners + gutter make it unmistakably a "sheet."

1c. Bottom Padding

The existing paddingBottom: 6rem (desktop) / 9.5rem (mobile) on the main scroll area is correct in code but visually insufficient when the nav bar is absent. Fix: ensure the MainSheet container also has bottom margin (mb-4) matching the top, so the sheet never touches the viewport bottom regardless of nav bar visibility.
