Green Goods · Handoff Bundle

Admin Cockpit Revamp

A four-surface admin tool for community gardens. This bundle hands the design off to a developer for implementation in React + Tailwind.

For the human handing this off: copy the contents of CLAUDE_CODE_PROMPT.md into Claude Code along with this zip. The prompt instructs it to audit your existing Storybook bottom-up and work tier by tier — no big-bang merges.
